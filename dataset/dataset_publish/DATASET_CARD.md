# 数据集卡片

## 1. 数据概览

- 名称：PsyQA TTS 情感增强语音数据（最终发布版）
- 样本总数：1500
- 语言：中文
- 格式：wav 音频 + txt 清单

## 2. 标签分布

- calm：764
- excited：736

## 3. 数据结构

- 音频目录：dataset_wavs/
- 清单文件：dataset_manifest.txt（制表符分隔）

字段定义：

- sample_id
- audio_file
- emotion
- intensity
- question_id
- text

## 4. 推荐用途

- 语音合成模型微调
- 情感语音建模与对比实验
- 语料分析与原型验证

## 5. 注意事项

- 本数据集包含合成增强样本。
- 情感标签由控制条件与模型生成过程共同决定，严格评测前建议抽检。
- 对外再分发前，请根据你的使用场景补充合规审查。
